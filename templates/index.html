About & How It Works

About the System

Cross-Language Visual Phishing Detection System is a cybersecurity solution that detects phishing emails using image-based analysis and OCR, independent of language. It extracts visual layouts, logos, and text features from email screenshots, converts them into vectors, and fuses them to compute an intent confidence score. The system classifies emails as phishing or legitimate based on the fused score. It supports multilingual detection and provides clear metrics for model performance.
